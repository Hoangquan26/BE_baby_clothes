## Description

Xây dựng bằng techstack: [NestJS] + [Prisma] (MySQL) + [Redis] + [RabbitMQ] + [Docker]
- Sẽ hỗ trợ lượng người dùng đồng thời cao
- Sẽ triển khai trong môi trường thực tế
## Project setup

```bash
$ pnpm install
```

## Compile and run the project

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Run tests

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
